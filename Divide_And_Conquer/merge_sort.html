Merge Sort

Merge sort is a sorting algorithm that uses a divide-and-conquer approach to sort an array or a list of elements. It works by breaking down the input list into smaller sub-lists, sorting those sub-lists recursively, and then merging them back together to produce a sorted output list.

How Merge Sort Works?

     
  1. Divide the unsorted list into n sublists, each containing one element (a list of one element is considered sorted).
  2. Repeatedly merge sublists to produce new sorted sublists until there is only one sublist remaining. This will be the sorted list.
  3. The merge operation works as follows:
    zeCompare the first element of each sublist.
    Move the smaller element to the new list.
    Repeat step 2 until one of the sublists is empty.
    Copy the remaining elements of the non-empty sublist to the new list.
  4. The merge sort algorithm works by recursively dividing the list into halves until we get to a point where there is only one element in each list. We then merge these lists to form a sorted list. The merging of the lists is done by repeatedly comparing the first element of each list and appending the smaller element to a new list.

Complexity Analysis of Merge Sort:

Time Complexity:
  • Best Case: O(n log n)
  • Worst Case: O(n log n)
  • Average Case: O(n log n)
Auxiliary Space:
O(n)because it requires additional memory to hold the sorted subarrays during the merging process.

Pseudo Code

function mergeSort(A: list of sortable items)
 if length(A) <= 1
  return A
 else
  mid := floor(length(A) / 2)
  left := mergeSort(A[0..mid-1])
  right := mergeSort(A[mid..length(A)-1])
  return merge(left, right)
 end if
end function

function merge(left: list of sortable items, right: list of sortable items)
 result := empty list
 while length(left) > 0 and length(right) > 0 do
  if left[0] <= right[0]

  else
   result.append(right[0])
   right.remove(right[0])
  end if
 end while
 if length(left) > 0
  result.extend(left)
 else
  result.extend(right)
 end if
 return result
end function
