Selection sort

Selection sort is a simple sorting algorithm that works by repeatedly finding the minimum element from an unsorted list and moving it to the beginning (or end) of the list. The algorithm maintains two sub-lists: the sorted sub-list and the unsorted sub-list. Initially, the sorted sub-list is empty, and the unsorted sub-list contains all the elements of the original list.

How Selection sort Works?

  1. Set the minimum index to 0.
  2. For each element in the unsorted sub-list:
  3. a. If the element is smaller than the element at the current minimum index, set the minimum index to the index of the smaller element.
  4. b. Swap the element at the minimum index with the first unsorted element in the sub-list.
  5. If the target element is less than the element at the middle index, set the high index to the middle index â€“1.
  6. Move the boundary of the sorted sub-list one element to the right.
  7. Repeat steps 2 and 3 until the entire list is sorted.

Complexity Analysis of Selection sort:

Time Complexity:
  • Best Case: O(n^2)
  • Worst Case: O(n^2).>
  • Average Case: O(n^2).
Auxiliary Space:
O(1) as except the variable to iterate through the list, no other variable is used.which means that the amount of memory used by the algorithm remains constant, regardless of the size of the input. This is because selection sort works by swapping elements in place, rather than creating new arrays or data structures to hold the sorted elements.

Iterative Method
    • function selectionSort(arr)
      n = length(arr)
      for i from 0 to n-2 do
      min_idx = i
      for j from i+1 to n-1 do
      if arr[j] < arr[min_idx] then
      min_idx = j
      end if
      end for
      if min_idx != i then
      swap(arr[i], arr[min_idx])
      end if
      end for
      return arr
      end function
