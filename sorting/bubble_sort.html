Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through a list of elements, compares adjacent elements and swaps them if they are in the wrong order. The algorithm gets its name from the way smaller elements "bubble" to the top of the list as it is sorted.

How Bubble Sort Works?

  1. Start at the beginning of the list.
  2. Compare the first two elements. If the first is greater than the second, swap them.
  3. Move to the next pair of elements, and so on, continuously making swaps as necessary.
  4. Continue until the end of the list is reached.
  5. Repeat the process for the entire list until no more swaps are needed.

Complexity Analysis of Bubble Sort:

Time Complexity:
  • Best Case: O(n) The best case occurs when the input list is already sorted. In this case, the algorithm only needs to traverse the list once, and no swaps are made. Therefore, the time complexity is O(n).
  • Worst Case: O(n^2) The worst case occurs when the input list is in reverse order. In this case, the algorithm needs to traverse the list n times, and each traversal involves n-1 comparisons and swaps. Therefore, the time complexity is O(n^2).
  • Average Case: O(n^2)The average case occurs when the input list is randomly ordered. In this case, the algorithm needs to traverse the list multiple times, and each traversal involves n-1 comparisons and swaps. Therefore, the time complexity is O(n^2).
Auxiliary Space:
O(1) Bubble sort is an in-place sorting algorithm, which means that it does not require additional memory to sort the input list. The space complexity is constant, O(1).

Pseudo Code

  • function bubbleSort(array)
    n = length(array)
    do
    swapped = false
    for i = 1 to n-1 do
    if array[i] > array[i+1] then
    swap(array[i], array[i+1])
    swapped = true
    end if
    end for
    n = n - 1
    while swapped
    end function
