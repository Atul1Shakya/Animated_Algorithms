Insertion sort

Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. It works by iterating over an array, and for each element, it finds the right position within the already sorted part of the array and inserts the element there. :

How Insertion sort Works?

  1. Assume the first element in the array is already sorted.
  2. For each remaining element in the array, starting from the second one:
  3. Compare the element with each of the elements in the already sorted part of the array, from right to left.
  4. If the element is smaller than the compared element, move the compared element one position to the right.
  5. Repeat until the element is in its correct position within the sorted part of the array.
  6. Repeat the above steps until all elements in the array are sorted. This algorithm is called "insertion sort" because it "inserts" each element into its correct position in the already sorted part of the array.

Complexity Analysis of Insertion sort:

Time Complexity:
  • Best Case: O(n)where n is the number of elements in the array. This happens when the input array is already sorted, and the algorithm simply performs a single pass over the array without swapping any elements.
  • Worst Case: O(n^2)where n is the number of elements in the array. This happens when the input array is sorted in reverse order, and the algorithm performs the maximum number of comparisons and element shifts for each element in the array.
  • Average Case: O(n^2)where n is the number of elements in the array. This is the expected running time of the algorithm when the input array is randomly ordered.
Auxiliary Space:
O(1) which means the algorithm uses a constant amount of extra memory to perform the sorting, regardless of the size of the input array.

Pseudo Code

  • procedure insertionSort(A: list of sortable items)
    for i from 1 to length(A) - 1 do
    key := A[i]
    j := i - 1
    while j >= 0 and A[j] > key do
    A[j + 1] := A[j]
    j := j - 1
    end while
    A[j + 1] := key
    end for
    end procedure
